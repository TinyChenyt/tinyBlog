# 登录页

在上一章的表单组件封装中我们新建了一个login页面，基于该页面实现登录

## 获取表单数据

在组件封装中，我们通过emit抛出了submit方法，在login中可以调用触发该方法

```vue
<script setup>
import { ref } from 'vue';
import TinyForm from '@/components/TinyForm.vue';

const formData = ref({
  username: '',
  password: ''
});

const formConfig = ref([
  {
    label: '用户名',
    type: 'input',
    placeholder: '请输入用户名',
    key: 'username',
    required: true
  },
  {
    label: '密码',
    type: 'input',
    placeholder: '请输入密码',
    key: 'password'
  }
]);

const submit = (data) => {
  console.log(data);
};
</script>

<template>
  <div>
    <TinyForm :form-data="formData" :form-config="formConfig" @submit="submit" />
  </div>
</template>
```

在开发中，通过请求登录接口，回显是否登录成功，然后实现登录用户信息的存储以及页面跳转

下面使用静态数据进行模拟请求

```js
// 测试账号
const testLoginInfo = ref({
  username: 'admin',
  password: '123456'
});

// 用户信息
const userInfo = ref({
  username: 'admin',
  sex: 'male',
  age: 18,
  address: '北京市',
  phone: '12345678901'
});

const submit = (data) => {
  login(data);
};

const login = (data) => {
  if (data.username === testLoginInfo.value.username && data.password === testLoginInfo.value.password) {
    ElMessage.success('登录成功');
    console.log(userInfo.value);
  }
  else {
    ElMessage.error('用户名或密码错误');
  }
};
```

成功之后，我们需要保存数据，并进行页面跳转，通过localStorage进行数据存储，然后跳转到home页面

首先我们对路由进行一下处理

```js
// 导入创建路由实例，和 history 模式
import { createRouter, createWebHistory } from 'vue-router';
import localStorage from '@/utils/localStorage';

// 路由规则
const routes = [
  // {
  //   path: '/', // 根路径
  //   name: 'home', // 路由名称
  //   component: () => import('../views/home.vue') // 路由组件
  // }
  {
    path: '/',
    name: 'login',
    component: () => import('@/views/login.vue')
  },
  {
    path: '/home',
    name: 'home',
    component: () => import('@/views/home.vue')
  }
];

// 创建路由实例
const router = createRouter({
  history: createWebHistory(),
  routes
});

// 路由导航守卫
router.beforeEach((to, from, next) => {
  // 未登录不允许访问其他页面
  if (to.path !== '/' && !localStorage.get('token')) {
    return next({ path: '/' });
  }

  // 防止重复登录
  if (localStorage.get('token') && to.path === '/') {
    return next({ path: from.path ? from.path : '/' });
  }

  next();
});

// 导出路由实例
export default router;

```

登录页面实现信息存储和跳转

```vue
<script setup>
import { ref } from 'vue';
import { ElMessage } from 'element-plus';
import { useRouter } from 'vue-router';
import localStorage from '@/utils/localStorage';
import TinyForm from '@/components/TinyForm.vue';

const router = useRouter();

const formData = ref({
  username: '',
  password: ''
});

const formConfig = ref([
  {
    label: '用户名',
    type: 'input',
    placeholder: '请输入用户名',
    key: 'username',
    required: true
  },
  {
    label: '密码',
    type: 'input',
    placeholder: '请输入密码',
    key: 'password'
  }
]);

// 测试账号
const testLoginInfo = ref({
  username: 'admin',
  password: '123456'
});

// 用户信息
const userInfo = ref({
  username: 'admin',
  sex: 'male',
  age: 18,
  address: '北京市',
  phone: '12345678901'
});

const submit = (data) => {
  login(data);
};

const login = (data) => {
  if (data.username === testLoginInfo.value.username && data.password === testLoginInfo.value.password) {
    ElMessage.success('登录成功');
    localStorage.set('token', '12345678901');
    localStorage.set('userInfo', userInfo.value);
    router.push({
      name: 'home',
      path: '/home'
    });
  }
  else {
    ElMessage.error('用户名或密码错误');
  }
};
</script>

<template>
  <div>
    <TinyForm :form-data="formData" :form-config="formConfig" @submit="submit" />
  </div>
</template>

<style scoped></style>
```
