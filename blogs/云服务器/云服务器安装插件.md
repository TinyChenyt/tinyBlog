# 云服务器安装插件

## 1.云服务器安装宝塔

镜像：Centos

### 1.1安装脚本：

```shell
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install.sh && sh install.sh
# 或者
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
```
