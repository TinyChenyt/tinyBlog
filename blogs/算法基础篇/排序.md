# 排序

## 冒泡排序

```js
const sort = (arr) => {
  let len = arr.length;

  for (let i = 1; i < len; i++) {
    let flag = true;

    for (let j = 0; j < len - i; j++) {
      if (arr[j] > arr[j + 1]) {
        let temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
        flag = false;
      }
    }
    if (flag) {
      break;
    }
  }
};
```

- 时间复杂度：最佳：O(n) ，最差：O(n2)， 平均：O(n2)

## 选择排序

```js
const sort = (arr) => {
  let len = arr.length;

  for (let i = 0; i < len; i++) {
    let minIndex = i;

    for (let j = i + 1; j < len; j++) {
      minIndex = arr[minIndex] > arr[j] ? j : minIndex;
    }
    if (minIndex !== i) {
      let temp = arr[i];
      arr[i] = arr[minIndex];
      arr[minIndex] = temp;
    }
  }
};
```

## 插入排序

```js
const sort = (arr) => {
  let len = arr.length;

  for (let i = 1; i < len; i++) {
    let compare = arr[i];

    let j;
    for (j = i - 1; j >= 0; j--) {
      if (arr[j] > compare) {
        arr[j + 1] = arr[j];
      } else {
        break;
      }
    }
    arr[j + 1] = compare;
  }
};
```

- 时间复杂度：最佳：O(n) ，最差：O(n2)， 平均：O(n2)
